# dotfiles

script to save dotfiles around my system and init scripts to run after reinstalling current OS

## flags (actions)

- s/b/c/p : save/backup/commit/push
- e   : edit config-files/init-scripts
- i   : run init scripts
- h   : help
- r   : remove backup directory

## flags (options)

- y   : auto answer yes [DANGEROUS]
- v   : verbose (show more info)
- d   : show diffs
