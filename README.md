# dotfiles-2.0

full rewrite of [autosaver script](https://github.com/daniele821/dotfiles)

## flags (actions)

- s/b : save/backup [xor]
- c/p : commit/push [or]
- e   : edit
- i   : initialize
- h   : help
- r   : rmdirs

## flags (options)

- y   : auto answer yes [DANGEROUS]
- f   : force allow dangerous operations [DANGEROUS]
- v   : verbose (show more info)
- d   : show diffs 
- u   : show untracked files
