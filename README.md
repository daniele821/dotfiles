# dotfiles

script to save dotfiles around my system and init scripts to run after reinstalling current OS

## flags (actions)

- s/b : save/backup [xor]
- c/p : commit/push [or]
- e   : edit
- i   : initialize
- h   : help
- r   : rmdirs

## flags (options)

- y   : auto answer yes [DANGEROUS]
- v   : verbose (show more info)
- d   : show diffs 
- u   : show untracked files
