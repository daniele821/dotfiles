# dbmain application

- script `dbmain.sh` does the operations necessary to run dbmain
- `bin` directory and `dbmain.sh` file MUST be in the same directory for `dbmain.sh` to work
- `tar xzf dbm-1102-linux-amd64-setup.tar.gz` should extract files in the correct place
- create a symlink to `dbmain.sh` in `~/.local/bin` to have it easily accessible
